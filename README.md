# pykli

ksqlDB command line client written in Python.

This project is in a very early stage. Please try and report bugs :)
PRs are welcome.

## Installation

```sh
pip install -U git+https://github.com/eshepelyuk/pykli@main
```

## Supported commands

* `SHOW ...` 
* `LIST ...`
* `DESCRIBE ...`
* `DROP ...`
